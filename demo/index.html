About Rembg Web

Rembg Web is a powerful TypeScript library that brings AI-powered background removal directly to web browsers. It supports multiple models, hardware acceleration via WebNN, and provides a complete toolkit for image processing.

  • Multiple Models: U2Net, ISNet, Silueta, and more
  • Hardware Acceleration: WebNN, WebGL, and CPU execution
  • Browser Compatible: Works in all modern browsers
  • TypeScript Support: Full type safety and IntelliSense
