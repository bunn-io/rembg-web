About Custom Models

This demo allows you to use your own U2Net-compatible ONNX model for background removal. The model should follow the U2Net architecture with the following requirements:

  • Input: RGB image tensor with shape [1, 3, H, W]
  • Output: Single channel mask tensor with shape [1, 1, H, W]
  • Input Name: Usually 'input.1' or 'input'
  • Normalization: Standard ImageNet normalization (mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])

You can upload your own model file or provide a URL to a hosted model.
